# Tailwind

Tailwind er et CSS-rammeverk som lar deg **designe** en nettside ved hjelp av forhåndsdefinerte klasser, i stedet for å skrive ren CSS-kode. Det gir utviklere et stort utvalg av fleksible, gjenbrukbare og responsive verktøy som gjør det enklere å bygge en stilfull og responsiv nettside eller applikasjon raskt. Tailwind inkluderer blant annet ferdige stiler for layout, tekstbehandling, farger, rammer og mye mer.

```html
<h1 className="text-4xl text-center font-bold">Foo Bar Baz</h1>
```

## Farger

Tailwind har et stort utvalg av farger som kan brukes i prosjektet. Fargene er delt inn i flere kategorier, som gråtoner, røde farger, blå farger, grønne farger, osv. Fargene kan brukes ved å legge til klassen `text-` etterfulgt av fargenavnet, for eksempel `text-red-500` eller `bg-gray-100`.

Les mer om farger i Tailwind [her](https://tailwindcss.com/docs/customizing-colors).

## Responsivt design

Tailwind har innebygd støtte for responsivt design. Dette gjør det enkelt å lage en nettside som ser bra ut på alle skjermstørrelser. For å bruke responsivt design i Tailwind, legger du til en av følgende prefikser foran klassen du vil gjøre responsiv:

- `sm:` - for skjermer som er 640px eller bredere
- `md:` - for skjermer som er 768px eller bredere
- `lg:` - for skjermer som er 1024px eller bredere
- `xl:` - for skjermer som er 1280px eller breder`
- `2xl:` - for skjermer som er 1536px eller breder`

Under er et eksmpel på hvordan tekststørrelsen kan gjøres responsiv:

```html
<h1 className="text-2xl md:text-4xl text-center font-bold md:text-red-400">Foo Bar Baz</h1>
```

<CodeExample>
  <h1 className="text-center text-2xl font-bold md:text-4xl md:text-red-400">Foo Bar Baz</h1>
</CodeExample>
